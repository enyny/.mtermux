# MyTermuxMustHave

Install [Termux-Monet](https://github.com/Termux-Monet/termux-monet/releases/download/v0.119.0-b1-36/termux-app_v0.119.0-b1-36+apt-android-7-github-debug_arm64-v8a.apk)

```console
curl -sL https://raw.githubusercontent.com/enyny/mytermuxsetup/refs/heads/main/run.sh | bash
```

Tide configure

```console
tide configure --auto --style=Lean --prompt_colors='True color' --show_time='24-hour format' --lean_prompt_height='Two lines' --prompt_connection=Dotted --prompt_connection_andor_frame_color=Lightest --prompt_spacing=Sparse --icons='Many icons' --transient=Yes
```
